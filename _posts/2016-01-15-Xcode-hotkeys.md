Xcode 的快捷键何其之多，这边只选取我自己使用频次最高的快捷键。主要是方便忘记的时候查询。

| Function         | Hotkey         |      |
| ---------------- | -------------- | ---- |
| 开启双编辑器           | ⌥ + ⌘ + Enter  |      |
| 关闭双编辑器           | ⌘ + Enter      |      |
| 右边导航窗口开关         | ⌘ + ⌥ + 0      |      |
| .m 与 .h 文件之间切换   | ⌘ + ⌃ + ↑ or ↓ |      |
| 按照浏览文件的前后顺序切换    | ⌘ + ⌃ + ← or → |      |
| 超强搜索，文件跳转        | ⌘ + ⇧ + o      |      |
| 文件内函数跳转，变量跳转     | ⌃ + 6          |      |
| 选中代码快左/右缩进       | ⌘ + [ or ]     |      |
| 选中代码块上/下移动一行     | ⌘ + ⌥ + [ or ] |      |
| 折叠/展开当前代码段       | ⌘ + ⌥ + ← or → |      |
| 跳转到左侧不同栏目        | ⌘ + 1 or 2..8  |      |
| 全局搜索             | ⌘ + ⇧ + f      |      |
| 跳转到目录树中当前打开文件的位置 | ⌘ + ⇧ + j      |      |
|                  |                |      |
|                  |                |      |
