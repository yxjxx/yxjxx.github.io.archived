博客的重点还是应该放在内容创作上，而不是网站是否好看。

现在这个基于 Jekyll 和 Github Pages 的 Blog 的主要问题：一是国内访问速度比较慢，二是 Jekyll 用起来相对复杂需要花比较多的精力在与写文章无关的地方。

所以换到了 MWeb 生成静态网站托管在国内的 coding.net 平台。MWeb 除了不好看和个人开发者新功能适配偏慢之外就没有太大的缺点了。
